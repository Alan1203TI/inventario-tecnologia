SISTEMA DE INVENTÁRIO SESI - ONEDRIVE + FIREBASE

COMO USAR
1. Extraia esta pasta no seu OneDrive.
2. Crie um projeto no Firebase: https://console.firebase.google.com/
3. Ative Authentication > Sign-in method > Email/password.
4. Ative Firestore Database.
5. Em Firestore Rules, cole o conteúdo do arquivo REGRAS-FIRESTORE.txt e publique.
6. No Firebase, vá em Configurações do projeto > Seus apps > Web app e copie a configuração.
7. Abra o arquivo firebase-config.js e substitua os valores COLE_AQUI pelos dados do seu Firebase.
8. Abra index.html no navegador.
9. Clique em Criar usuário na primeira vez.
10. Depois de entrar, clique em “Enviar dados iniciais para Firebase” somente uma vez.

OBSERVAÇÕES
- Os arquivos podem ficar no OneDrive.
- Os dados ficam online no Firebase.
- Alterou em um computador, aparece em outro após login.
- Use Exportar CSV/Excel para gerar relatório.
- Use Backup JSON para uma cópia completa dos dados.
