Configurações e backup

Use “Enviar dados iniciais” apenas uma vez, após configurar o Firebase. Esta área fica visível apenas para Administrador.

Imagens padrão por categoria e modelo

Cada categoria agora pode ter modelos próprios com uma imagem específica. No cadastro, ao selecionar o tipo e o modelo, o sistema usa a imagem daquele modelo. Se o modelo não tiver imagem, ele usa a imagem padrão da categoria.

Gerar etiquetas com QR Code em PDF

Selecione o tipo de equipamento para gerar um PDF com QR Code, patrimônio, hostname e número de série. O arquivo fica pronto para imprimir, recortar e colar nos equipamentos.

Alterar status por local

Selecione um local e aplique um novo status para todos os equipamentos cadastrados nesse local. A alteração será registrada no histórico.

Importar equipamentos em massa por Excel

Envie uma planilha .xlsx, .xls ou .csv. O sistema reconhece colunas como Tipo, Categoria, Modelo, Nº de série, Patrimônio, IMEI, Hostname, Local, Etiqueta, Responsável, Status, Origem e Observação.
